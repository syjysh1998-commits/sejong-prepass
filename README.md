# 세종 민원 프리패스 (Pre-Pass) 목업

세종특별자치시 읍면동 주민센터 민원 사전준비 시스템의 인터랙티브 목업입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Vercel 배포

1. GitHub에 이 저장소를 push합니다
2. [vercel.com](https://vercel.com)에서 "Import Project"로 연결합니다
3. Framework Preset: **Vite** 선택
4. Deploy 클릭

자세한 배포 가이드는 제작자에게 문의하세요.
